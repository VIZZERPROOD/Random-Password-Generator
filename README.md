## Описание
GUI-приложение на Python Tkinter для генерации случайных паролей с настраиваемыми параметрами.

Возможности:
- Ползунок длины пароля (от 4 до 64)
- Выбор наборов символов:
  - цифры
  - буквы
  - спецсимволы
- Генерация пароля через `random`
- История генераций в таблице
- Сохранение/загрузка истории в `password_history.json`
- Копирование сгенерированного пароля в буфер обмена

## Запуск
```bash
cd RandomPasswordGenerator
python3 main.py
```

## Пример использования
1. Установите длину пароля ползунком.
2. Отметьте нужные типы символов.
3. Нажмите `Сгенерировать`.
4. При необходимости нажмите `Скопировать`.
5. Сохраните историю кнопкой `Сохранить историю`.
